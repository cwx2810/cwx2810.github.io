搭建hexo博客

搭建前的准备工作

1.安装git for windows

git是基础，关于博客所有的操作都基于git bash

2.安装node.js

hexo博客基于node.js

3.安装markdown2

markdown2可以方便的修改博客内容

4.安装sublimetext

sublimetext用于修改一些配置文件

5.创建github账号

有了github账号才能将博客发布到互联网

搭建过程

在你喜欢的位置新建hexo文件夹，在文件夹内右键选择git bash，输入以下命令安装hexo

至此本地博客搭建好，执行以下命令可在浏览器输入localhost:4000进行本地查看

在github创建新的仓库，名字是：你的账户名.github.io
在hexo文件夹中找到_config.yml，用sublimetext打开，在最后添加以下代码

deploy: 
  type: git
  repository: https://github.com/你的账户名/你的账户名.github.io.git
  branch: master

修改以下代码

title: 站点名
subtitle: 副标题
description: 对站点的描述，给搜索引擎看的
author: 作者，在站点左下角可以看到

需要注意的是所有冒号后面都有一个英文空格
执行下列指令即可发布

需要输入github账户名和密码，密码是不显示的
现在登陆 http://你的账户名.github.io 即可访问博客
至此，博客搭建完毕

##发布博文

执行以下命令

在hexo\source_posts中打开这个文件，在开头添加如下代码

title: 标题
data: 发表日期
categories: 分类
---
这里用markdown2写正文

执行hexo g和hexo d同步到github

命令说明

hexo g == hexo generate
hexo d == hexo deploy
hexo s == hexo server
hexo n == hexo new
